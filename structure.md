### 14.2 設計方針

- **安定性最優先**: 入力が壊れない/落ちない/状態が迷子にならない
- **自動確定禁止**: 提案は OK、確定はユーザー操作のみ
- **非同期処理**: Index/Build は UI をブロックしない
- **PDF 保護**: ビルド失敗時は前回成功 PDF を保持
- **ログ洪水禁止**: Issues が一次窓口

## 技術構成（参考）

```
tex64/
├── electron/              # Electron メインプロセス
│   ├── main.cjs           # エントリ・ウィンドウ・IPC
│   ├── preload.cjs        # Bridge（window.tex64Bridge）
│   ├── handlers/          # IPC ハンドラ群
│   └── services/          # バックエンドサービス
│       ├── build.cjs      # latexmk ビルド
│       ├── formatter.cjs  # latexindent 整形
│       ├── indexer.cjs    # ドキュメントインデックス
│       ├── search.cjs     # テキスト検索
│       ├── synctex.cjs    # SyncTeX
│       ├── pdf.cjs        # PDF ウィンドウ管理
│       ├── agent.cjs      # AI エージェント
│       └── ...
│
├── web-src/               # Web UI ソース（TypeScript）
│   ├── main.ts            # エントリ
│   └── app/               # 機能モジュール
│       ├── editor-*.ts    # エディタ関連
│       ├── blocks/*.ts    # ブロック/MathLive 関連
│       ├── ai-*.ts        # AI 関連
│       └── ...
│
└── Resources/web/         # Web UI 成果物 + 静的資産
    ├── index.html         # メイン HTML
    ├── theme.css          # スタイル
    ├── main.js            # ビルド生成物
    ├── monaco/            # Monaco Editor
    ├── mathlive/          # MathLive
    └── pdfjs/             # PDF.js
```
