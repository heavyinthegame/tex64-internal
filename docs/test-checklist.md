# tex180 機能確認手順（網羅版）

## 0. 事前準備
- [OK] `latexmk` がインストール済みである（未導入の場合はビルドで「latexmk が見つかりません」になる想定）
- [OK] 書き込み可能な場所にテスト用フォルダを作成できる
- [OK] `.tex180` を確認する場合は隠しファイル表示を有効にしておく
- [OK] アプリを起動できる（Xcode実行でもビルド済みアプリでも可）

## 1. 起動・ランチャー（新規/既存）
- [OK] 起動直後に「新規プロジェクト作成 / 既存フォルダを開く」だけが表示される
- [OK] テンプレート選択が「論文 / 講義ノート」で切替できる
- [OK] 「新規プロジェクト作成」でフォルダ選択が出る（書き込み可能なフォルダ）
- [OK] 作成直後に編集画面へ遷移し、ファイル選択ボタンが表示されない
- [OK] 編集画面に入った時点で `main.tex` などのTeXが自動で開かれ、すぐ入力できる
- [OK] 「既存フォルダを開く」でも編集画面へ遷移する

## 2. 新規プロジェクト（テンプレ生成）
- [OK] 論文テンプレで `main.tex` が生成され、`title/author/abstract/section` が入る
- [OK] 講義ノートテンプレで `main.tex` が生成され、`目的/内容/まとめ` が入る
- [OK] 生成後、`main.tex` を開いて内容が一致する
- [OK] `.tex180` はこの時点では作られていない（必要時のみ）

## 3. 既存フォルダ（ワークスペース）
- [OK] 既存フォルダを開くと、左のエクスプローラーにファイル一覧が出る
- [OK] ワークスペース名が左上（エクスプローラー内）に表示される
- [OK] `.git` / `node_modules` などは一覧から除外される
- [OK] ミニアウトライン（上部）に現在のファイル名が反映される

## 4. ファイル操作（開く/保存/折りたたみ）
- [OK] ファイルをクリックして開ける（未保存変更がある場合は確認ダイアログ）
- [OK] 編集すると上部パンくずに「●」が付く
- [OK] 保存ボタンまたは `⌘S` で保存でき、「●」が消える
- [OK] 新規ファイル/新規フォルダ作成ができ、ツリーが更新される
- [OK] エクスプローラー上部に新規ファイル/新規フォルダボタンが表示されない
- [OK] フォルダの折りたたみ状態が保持され、再起動後も維持される

## 5. メインTeX（自動検出/手動指定）
- [OK] 設定タブに「メインTeX」セレクトと「自動検出/自動に戻す」ボタンがある
- [OK] 自動検出で `main.tex` が選ばれる（該当がない場合は未選択）
- [OK] セレクトで別の `.tex` を指定すると手動指定として保存される
- [OK] ボタンが「自動に戻す」に変わり、押すと自動検出に戻る
- [OK] 手動指定したファイルがビルド対象になる（後述のビルドで確認）

## 6. インデックス/アウトライン（章節・図表・TODO）
- [OK] `main.tex` に `\section` / `\subsection` を追加し保存する
- [OK] `\begin{figure}` + `\caption{...}` を追加し保存する
- [OK] `\begin{table}` + `\caption{...}` を追加し保存する
- [OK] `\todo{...}` または `TODO: ...` を追加し保存する
- [OK] アウトラインタブに「章節/図/表/TODO/ラベル/参考文献」が表示される
- [OK] 各項目をクリックすると該当行にジャンプし、行がハイライトされる

## 7. 参照・引用（補完/ジャンプ）¥¥¥
- [ ] `\label{sec:intro}` を追加し保存する
- [ ] `\ref{` と入力すると `sec:intro` が補完候補に出る
- [ ] `.bib` ファイルを作成して `@article{smith2020,...}` を追加・保存
- [ ] `\cite{` と入力すると `smith2020` が補完候補に出る
- [ ] アウトライン「ラベル/参考文献」からクリックでジャンプできる

## 9. Auto Build
- [ ] 上部の自動ビルドボタンと設定のスイッチが連動する
- [ ] Auto Build ONで `.tex` を編集→保存するとビルドが自動実行される
- [ ] Auto Build OFFなら保存してもビルドされない

## 10. Quick Insert（挿入UX）

- [OK] `⌘Z` で取り消せる

## 11. ブロック（数式/表 + MathLive）

- [ ] MathLiveが読める環境なら数式フィールドに変わる（未読込ならテキスト）
- [ ] 数式を入力→プレビュー→確定で `\[ ... \]` が挿入される
- [ ] 「表」で行/列指定→プレビュー→確定で `tabular` が挿入される
- [ ] ブロック一覧に追加され、クリックで再編集できる

- [OK] クリックで該当ファイル/行にジャンプできる
