                        コンパイルエンジン
                        ビルドに使用するエンジンを選択します。

                        ビルドプロファイル
                        プロジェクトの流儀に合わせて latexmk の出力先や追加オプションを切り替えます。
                      

                      

                        
                      

                    

                    

                      

                        プロファイル名
                      

                      

                        
                      

                    

                    

                      

                        outDir
                        生成物の出力先（空ならデフォルト）。例: build
                      

                      

                        
                      

                    

                    

                      

                        latexmk 追加オプション
                        例: -use-biber -shell-escape
                      

                      

                        
                      

                    

                    

                      
                      
                    

                    

                      Default は tex64 の標準設定です。プロジェクト固有の outDir や biber/shell-escape が必要な場合はプロファイルを作成してください。
                    

                    

                      

                        clean
                        補助ファイル（.aux/.log/.synctex.gz 等）を削除します。
                      

                      

                        
                        
                      

                    

                    

                      clean -C は PDF なども削除します。実行前に確認ダイアログが表示されます。

                        latexmk
                        ビルドを自動化するコマンド。TeX Distribution に同梱されることが多い。

                        latexindent
                        整形（Format）に使用。TeX Distribution に同梱されることが多い。
                      

                      

                        確認中...
                        
                      

                    

                    

                      

                        ChkTeX
                        lint（文章/LaTeXの静的チェック）に使用。TeX Distribution に同梱されることが多い。
                      

                      

                        確認中...
                        
                      

                    

                    

                      

                        SyncTeX
                        PDF へのジャンプに使用。TeX Distribution に同梱されることが多い。
                      

                      

                        確認中...
                        
                      

                    

                    

                      本アプリのビルド/整形/SyncTeX には TeX Distribution と
                      latexmk / latexindent / chktex / synctex が必要です（多くは同梱）。
